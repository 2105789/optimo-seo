Writo AI Scraper Documentation

A complete guide to setting up and configuring the Writo AI Scraper WordPress plugin.

Installation

  1. Download the writo-ai-scraper.zip file.
  2. Log in to your WordPress Admin Dashboard.
  3. Navigate to Plugins → Add New → Upload Plugin.
  4. Upload the zip file and click Install Now.
  5. Once installed, click Activate.
  6. Go to the new Writo AI menu item in your admin sidebar.

Requirements

  • WordPress 6.0 or higher
  • PHP 7.4 or higher
  • A valid Google Gemini API Key
  • Outbound HTTP connections allowed on your server

API Key Setup

You need a Google Gemini API key from Google AI Studio.

  1. Go to Google AI Studio.
  2. Sign in with your Google account.
  3. Click Create API Key.
  4. Copy the key and paste it into the Gemini API Key field in the plugin settings.
  5. Click Test API to verify the connection.

Tip: Free tier API keys have usage limits. For heavy usage, consider a paid plan.

Gemini Models

The plugin supports multiple Gemini models:

  • gemini-3-pro-preview - Most intelligent, best for complex rewriting.
  • gemini-3-flash-preview - Fast and capable.
  • gemini-2.5-pro - Stable, reliable performance.
  • gemini-flash-latest - Optimized for speed.
  • gemini-flash-lite-latest - Fastest, lowest cost.

Categories

In the Allowed Categories section, select which WordPress categories the AI can assign to generated posts. The AI will analyze the content and choose the most relevant categories from your selection.

Note: If you don't select any categories, posts will default to "Uncategorized".

Source URLs

Enter the website URLs you want to scrape. Add one URL per line in the Source URLs textarea.

https://news.example.com/
https://blog.anothersite.org/tech/

These can be homepages (for hub mode) or direct article URLs.

Scraping Options

Scrape Depth

  • 1 (URL Only): Scrapes only the exact URL provided.
  • 2 (Hub Mode): Finds article links on the source page and scrapes those linked articles.

Max Posts Per Source

Limits how many articles are generated from each source URL per run. Useful for balancing content from multiple sources.

Global Posts Limit

A safety cap on the total number of posts created across all sources in a single run.

Images

When enabled, the plugin extracts images (including OG:Image and lazy-loaded images) and sets the first valid one as the Featured Image.

Automation

Schedule automatic runs using WP-Cron:

  • Hourly
  • Twice Daily
  • Daily
  • Weekly

Use the Write Now button to trigger an immediate manual run.

Topic Keywords (Optional)

Filter content by keywords. Enter comma-separated keywords (e.g., bitcoin, crypto, blockchain). Only articles containing at least one of these keywords in their title or content will be processed.

Uninstallation

The plugin provides a clean uninstall option:

  1. In the settings, check "Delete all plugin settings and data upon uninstallation".
  2. Deactivate the plugin.
  3. Delete the plugin via the WordPress Plugins page.

All plugin options will be removed from the database. Generated posts are not deleted, as they belong to you.

Troubleshooting

Posts Not Being Generated

  • Verify your API key is correct.
  • Check if the source website blocks bots (some sites return empty content).
  • If using Keywords, ensure the scraped content actually contains those keywords.
  • Enable WP_DEBUG_LOG in wp-config.php and check wp-content/debug.log for detailed error messages from the plugin.

Duplicate Posts

The plugin stores the source URL as post meta (writo_source_link) and checks for duplicates before creating new posts. If you see duplicates, ensure the previous post wasn't deleted from the database incorrectly.
